refresh-token 토큰 재발급

refresh-token 재발급을 처리하기 위한 API입니다.

API 기본 정보#

요청 URL : /user/refreshtoken/

메서드 : GET

인증 : YES

Permissions required : None

응답 결과#

  • 상태 코드 : 200 OK
필드타입설명
responsestring응답 결과
messagestring응답 메세지
datastring응답 데이터
